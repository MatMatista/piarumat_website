1.8-StylePvP

Questo plugin aggiunge il sistema di PvP in stile 1.8 nei mondi che scegli tu all'interno del tuo server. Con un config semplicissmo da personalizzare offre la possibilità di creare modalità dai combattimenti spettacolari.

PerfectLB

Questo plugin permette di creare Leaderboards 100% personalizzabili per classifiche nel tuo server minecraft. Il plugin include top 10 per kills, deaths playtime e altro, per tener conto sempre di chi comanda nel server.
